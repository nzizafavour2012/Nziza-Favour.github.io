# Nziza Favour Portfolio

A modern, responsive portfolio website showcasing Nziza Favour's software development skills and projects.

## Quick Start
1. Open `index.html` in your browser (double-click or drag to browser).
2. Toggle dark/light theme.
3. Navigate sections.

## Features
- Responsive design (mobile-first)
- Dark/Light theme toggle (persists)
- Smooth scrolling navigation
- Skills showcase (Python, JS, ML, Blockchain, etc.)
- GitHub-integrated projects
- Contact form (demo)

## Customization
- Edit `index.html` for content/projects
- Modify `styles.css` for design
- Update `script.js` for functionality

## Deployment
- GitHub Pages: Create repo, enable Pages
- Netlify/Vercel: Drag-drop folder

Built with pure HTML/CSS/JavaScript 🚀
